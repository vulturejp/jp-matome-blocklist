# Japanese Matome Blog Blocklist

Japanese Matome Blog Blocklist provides blocklists for Japanese 5ch/2ch matome blog domains.

## Lists

- `urls.txt`: plain URL list
- `ublock.txt`: uBlock Origin filter list
- `adguard.txt`: AdGuard filter list
- `ublacklist.txt`: uBlacklist rule list
- `sites.json`: machine-readable site metadata

## Subscribe

Use the raw GitHub URLs below:

